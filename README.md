# pierangelo.me

Personal blog and portfolio — a collection of my work and thoughts on **Digital Marketing**, **Photography & Video**, and **Board Game Design**.

🌐 [pierangelo.me](https://pierangelo.me)

---

## ✨ What's inside

- **Digital Marketing** — projects, strategies, and reflections on the digital marketing world
- **Photo & Video** — a collection of my photography and video work
- **Boardgames** — designing and prototyping board games as a passion project

## 📄 Pages

- **Home** — intro and latest posts
- **Posts** — all blog articles
- **About** — who I am
- **Search** — full-text static search across all content

## 🛠️ Tech Stack

- [Astro](https://astro.build/) — main framework
- [AstroPaper](https://github.com/satnaing/astro-paper) — base theme (customized fork)
- [TailwindCSS](https://tailwindcss.com/) — styling
- [Pagefind](https://pagefind.app/) — static search
- [Cloudflare Pages](https://pages.cloudflare.com/) — hosting and deployment

## 🚀 Running Locally

```bash
pnpm install
pnpm dev
```

The site will be available at `http://localhost:4321`.

## 📦 Build & Deploy

Deployment happens automatically on Cloudflare Pages on every push to the `main` branch.

To build locally:

```bash
pnpm build
pnpm preview
```
